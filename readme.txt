GetResponse Forms by Optin Cat is a fast, powerful and convenient tool that makes it easy for anyone to create beautiful GetResponse forms for their blogs. In minutes you can create your forms and start increasing your conversions!

= GetResponse Forms by Optin Cat – Types of Optins =

GetResponse Forms by Optin Cat supports a number of different forms, giving you full flexibility when it comes to the way you integrate your signup option. 

These include:

*   GetResponse Post Boxes. These are forms you can insert into any page or post on your blog.
*   GetResponse Sidebar Widgets.
*   GetResponse Lightbox Popups. This is the most effective way to expand your GetResponse list because users can't miss it.

== Installation ==

1. Upload the plugin to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. In your sidebar, select 'Optin Cat -> Add New' to create a new table
